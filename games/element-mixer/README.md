# Element Mixer 🧪

A crafting/alchemy game inspired by Infinite Craft, where you combine elements to discover new ones!

## 🎮 How to Play

1. **Start with 4 basic elements**: Water 💧, Fire 🔥, Wind 🌪️, Earth 🌍
2. **Drag elements** from the sidebar into the workspace
3. **Drag one element onto another** to combine them
4. **Discover new elements** and unlock 150+ unique combinations!

## ✨ Features

- **150+ Recipes**: Combine elements to create everything from Steam to Dragons
- **Discovery Tracking**: See your newly discovered elements marked as "NEW!"
- **Recent Discoveries Panel**: Track your latest 10 discoveries with timestamps
- **Search & Sort**: Find elements quickly with search and sort by name or time
- **Persistent Saves**: Your progress is automatically saved to localStorage
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Visual Feedback**: Animated popups show your creations
- **Emoji Icons**: Every element has a unique emoji for easy identification

## 🎨 Game Controls

### Desktop
- **Drag & Drop**: Click and drag elements from sidebar to workspace
- **Combine**: Drag one workspace element onto another
- **Search**: Type in the search box to filter elements
- **Sort**: Click "By Time" or "By Name" to sort your elements

### Mobile/Tablet
- **Tap**: Tap an element in sidebar to add it to workspace
- **Drag**: Touch and drag elements around the workspace
- **Combine**: Drag one element onto another to combine

### Buttons
- **Clear Workspace**: Remove all elements from workspace (keeps discoveries)
- **Reset Game**: Reset all progress back to the 4 starting elements

## 💾 Save System

Your progress is automatically saved to your browser's localStorage:
- All discovered elements are saved
- Recent discoveries with timestamps are saved
- Progress persists across browser sessions

## 🎓 Tips & Tricks

1. **Start Simple**: Try combining all basic elements with each other first
2. **Branch Out**: Once you discover Steam, Cloud, Stone, etc., try combining them
3. **Think Logically**: Many combinations make sense (Rain + Sun = Rainbow)
4. **Experiment**: Some combinations are surprising!
5. **Use Search**: If you have many elements, search to find what you need
6. **Track Discoveries**: Watch the Recent Discoveries panel for inspiration
7. **Clear Workspace**: Use the clear button to organize when workspace gets messy

## 🎮 Differences from Infinite Craft

**Element Mixer** is inspired by Infinite Craft but with key differences:

1. **Predefined Recipes**: Uses a curated recipe database instead of AI
2. **No External API**: Works completely offline, no internet needed
3. **Faster**: Instant combinations, no waiting for API responses
4. **Free**: No API costs or rate limits
5. **Consistent**: Same combinations always work the same way
6. **Educational**: Learn logical element combinations
